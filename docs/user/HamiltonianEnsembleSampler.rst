Hamiltonian Ensemble Sampler
____________________________

A Hamiltonian sampler is a Markov Chain Monte Carlo (MCMC) algorithm 
that uses concepts from Hamiltonian dynamics to explore probability distributions 
more efficiently than simple random-walk methods.
